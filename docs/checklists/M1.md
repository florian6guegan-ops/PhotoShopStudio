# M1 — à faire sur place avec Florian

## Capture DEVMODE DS620 (écran Catalogue)
- [ ] DS620 allumée, papier 10×15 chargé
- [ ] Studio Photo → Catalogue → produit 10×15 (imprimante = DNP DS620) → « Capturer réglages »
- [ ] Dans le dialogue DNP : format 10×15 (PC), finition brillante, sans marges, correction couleur pilote par défaut
- [ ] Vérifier « réglages pilote : capturés ✓ » sur la fiche produit

## Tirage test DS620 (consomme 1 feuille)
- [ ] Importer une carte SD ou un dossier de test
- [ ] Sélectionner 1 photo, produit 10×15 DS620, recadrer si besoin
- [ ] Imprimer — vérifier : bords pleins (pas de marge), pas d'étirement, couleurs correctes
- [ ] Poser une règle : l'image doit faire exactement 102×152 mm
- [ ] Vérifier la commande dans « Commandes du jour » (statut Imprimée)

## Recette écran opérateur
- [ ] SD détectée < 5 s après insertion
- [ ] 500 photos parcourues sans blocage
- [ ] Tuer l'appli pendant une impression → au redémarrage, invite « Impression confirmée ? » (jamais de doublon auto)
